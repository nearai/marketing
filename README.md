# NEAR AI Marketing

The landing and marketing pages for NEAR AI (blogs, RFPs, etc).

This project is built with Next JS and [NEAR AI UI](https://github.com/nearai/ui).

## Local Development

```shell
cp .env.example .env
nvm use
npm i -g pnpm
pnpm i
pnpm dev
```

## Publishing

Pushing to the `main` branch will deploy via Vercel.
